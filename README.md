# Scientific Calculator

A Python CLI scientific calculator.

## Usage

```bash
# Basic operations
python calculator.py + 2 3      # Result: 5.0
python calculator.py - 10 4    # Result: 6.0
python calculator.py * 3 4     # Result: 12.0
python calculator.py / 10 2   # Result: 5.0

# Scientific operations
python calculator.py sqrt 16       # Result: 4.0
python calculator.py sin 90        # Result: 1.0
python calculator.py cos 0         # Result: 1.0
python calculator.py log 100       # Result: 2.0
python calculator.py ln 2.718     # Result: ~1.0
python calculator.py factorial 5  # Result: 120.0

# Interactive REPL
python calculator.py --repl
```

## Install

```bash
pip install -r requirements.txt
```
